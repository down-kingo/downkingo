## Status Atual (v3.0.0)

O DownKingo v3 e uma reescrita completa sobre o Wails v3, com novas funcionalidades e melhorias significativas em relacao a v2.

### Implementado na v2

- Download de video do YouTube e 1000+ sites
- Selecao de qualidade/resolucao
- Instalacao automatica de yt-dlp e FFmpeg
- Interface com tema Dark/Light
- Cross-platform (Windows, Linux)
- Fila de downloads concorrentes
- Monitor de clipboard inteligente
- Historico persistente (SQLite)
- Conversores de video, audio e imagens
- Internacionalizacao (5 idiomas)
- Roadmap interativo com votacao via GitHub
- Autenticacao GitHub (Device Flow)
- Deep Links (`kingo://`)
- Auto-update via GitHub Releases
- Onboarding interativo
- Customizacao de aparencia (accent color, densidade)

### Novo na v3

- Migracao completa para Wails v3
- Sistema de build via Taskfile
- Bindings type-safe auto-gerados
- Transcritor de audio/video via Whisper (local, offline)
- Sistema de votacao exclusiva no Roadmap (up/down)
- Sincronizacao de reacoes com GitHub
- Testes automatizados (Go + Vitest)
- Documentacao de arquitetura e ADRs
